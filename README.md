# LinkedIn Clone - MERN Project

This project is a LinkedIn-like social networking platform built using the MERN stack (MongoDB, Express, React, Node.js).

## Project Structure

- `backend/` – Node.js and Express server, API routes, and database models.
- `frontend/` – React frontend for UI, pages, and components.

## Features

- User authentication and profiles
- Connect with other users
- Create, like, and comment on posts
- Real-time notifications (optional)
- Responsive UI
